# Alexander Ang — Portfolio

Personal portfolio site built with Vue 3 and Vite. Live at [alexanderang.vercel.app](https://alexanderang.vercel.app).

## Tech Stack

- **Vue 3** — Composition API with `<script setup>`
- **Vue Router** — client-side routing
- **Vite** — build tooling
- **Vercel** — hosting and deployment

## Features

- Light / dark theme toggle
- Writing page with markdown post rendering
- Smooth page transitions
- Responsive layout

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
